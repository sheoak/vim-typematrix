INTRODUCTION                                    *typematrix*

This plugin set a few mappings that are easy to access on typematrix

**Under developpment**


MAPPINGS                                        *typematrix-mappings*

    <Backspace>     Page up
    <Space>         Page down
    <Up>            Wrap up (gs)
    <Down>          Wrap down (gt)
